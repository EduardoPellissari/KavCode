# Site KavCode

Site institucional com seção de portfólio, criado a partir do brandbook e da logo da KavCode.

## Arquivos

- `index.html`: estrutura da página.
- `styles.css`: identidade visual, responsividade e componentes.
- `script.js`: menu mobile, filtros do portfólio e resposta do formulário.
- `assets/kavcode-logo.png`: logo usada no site.
- `assets/brandbook-kavcode.pdf`: brandbook original de referência.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Deploy no Render

Este projeto está configurado para rodar como Static Site no Render.

1. Suba a pasta para um repositório no GitHub.
2. No Render, crie um novo serviço usando a opção `Blueprint`.
3. Selecione o repositório do site.
4. O Render vai ler o arquivo `render.yaml` e publicar a pasta raiz.

Também é possível criar manualmente como `Static Site`:

- Build Command: deixe vazio.
- Publish Directory: `.`
